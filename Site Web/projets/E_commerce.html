E-commerce

Créer une application e-commerce moderne, intuitive et responsive, permettant aux utilisateurs de naviguer, rechercher, acheter des produits et gérer leurs commandes.

Technologies utilisées

  • Frontend :
    • Angular: un framework pour clients, open source, basé sur TypeScript , Il permet la création d’applications Web
  • Backend :
    • Node.js : Pour construire des API REST sécurisées et performantes.
  • Base de données :
    • MongoDB : Pour stocker les informations des utilisateurs, des hôtels, des chambres et des réservations.
