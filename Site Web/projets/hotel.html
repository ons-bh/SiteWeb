Hotel Booking

Créer une application web permettant aux utilisateurs de rechercher, réserver et gérer leurs séjours dans des hôtels,ainsi les orders tout en offrant aux administrateurs la possibilité de gérer les chambres, les réservations .

Technologies utilisées

  • Frontend (Interface utilisateur) :
    • React.js : Pour créer une interface utilisateur dynamique et réactive. Bibliothèques complémentaires : React Router (navigation), Axios (requêtes API).
  • Backend (Serveur) :
    • Node.js : Pour construire des API REST sécurisées et performantes.
  • Base de données :
    • MongoDB : Pour stocker les informations des utilisateurs, des hôtels, des chambres et des réservations.
