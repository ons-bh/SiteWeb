Gestion de Projet

Ce projet vise à créer une application web permettant de gérer des projets en utilisant les méthodologies agiles Scrum et Kanban , avec des fonctionnalités dédiées à la planification, au suivi et à la collaboration d'équipe

Technologies utilisées

  • Frontend :
    • Angular: un framework pour clients, open source, basé sur TypeScript , Il permet la création d’applications Web
  • Backend :
    • Spring Boot :Framework Java pour la création d'API REST sécurisées.
  • Base de données :
    • MySQL :Stockage des données persistante
