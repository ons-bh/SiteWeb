Stage de PFE

  • Duree:

    • Janvier-juin 2024

  • Societe :

    • Primatec

  • Objectif :

    • creer une platforme pour gestion des projets on choisit une des methodes d'agile selon le besoin en utilisant le framework angular pour le frontend et le framework SpringBoot pour le backend

  • Description :

    • Ce projet consiste à développer une plateforme de gestion de projets en ligne basée sur les principes des méthodes agiles. La plateforme permettra aux équipes de gérer leurs projets en utilisant une méthodologie Agile adaptée, telle que Scrum, Kanban ou Lean, en fonction des besoins spécifiques du projet. Le frontend sera développé avec le framework Angular pour une interface utilisateur dynamique et réactive, tandis que le backend sera géré avec Spring Boot pour une gestion efficace des données et des processus métier.
